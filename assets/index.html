HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much spave we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Asigning colors in CSS can be done in many ways, using referances such as "gray" or "blue". We can also use a colors hex #
  • CSS is more or less the clothing or the outward appearance a user of the site would see, it allows you to dress up your webpage as you desire

Git

  • Git is one of the leading names behind the coding community
  • Git was first created in 2005, for the development of the Linux Kernel!
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is read from top to bottom, therefore the order of code affects the outcome.
  • JavaScipt allows the users to interact with websites!
